# HALOS Project Stats

Periodic snapshots of project health and delivery metrics. One file per snapshot date.

## Fields

| Field | Description |
|---|---|
| `snapshot_date` | Date of this snapshot |
| `project_start` | Date of first commit |
| `days_since_start` | Calendar days elapsed |
| `repos` | Commit counts per repo |
| `spec` | halos-spec artifact counts |
| `proposals` | Proposal counts by status |
| `site` | halos site content counts |
| `tooling` | Agent and developer tooling counts |
| `community` | Signatory count |
| `issues` | Open/closed issue counts per repo |
