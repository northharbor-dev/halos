# Homepage visual concepts

Concept illustrations for the HALOS homepage — "community + encompassing framework" theme.

| File | Description |
|------|-------------|
| `halos-community-encompassed.png` | Diverse community (men, women, children) with luminous orbits and aura |
| `halos-person-encompassed-concept-a.png` | Single person with luminous orbital ring/halo |
| `halos-person-encompassed-concept-b.png` | Concentric rings radiating from human at center |
| `halos-person-encompassed-concept-c.png` | Flowing arcs wrapping around person |

To switch which concept appears on the homepage, update the `src` and `srcset` in `index.md` in the `.home-visual` section.
