# HALOS Identity Concepts

This folder contains early public concept exploration for the HALOS visual identity.

The goal is not to over-brand the framework, but to explore visual directions that are consistent with HALOS as an open, principled, governance-aware framework rather than a product.

## Concepts

- `halo-ring-prompts.md` — prompt set for the Halo Ring direction
- `tri-layer-orbit-prompts.md` — prompt set for the Tri-Layer Orbit direction
- `halo-ring-concept.svg` — simple vector sketch for the Halo Ring direction
- `tri-layer-orbit-concept.svg` — simple vector sketch for the Tri-Layer Orbit direction

## Interpretation

- **Halo Ring** emphasizes structure, scope, governance, and durable framework identity.
- **Tri-Layer Orbit** emphasizes relationship among distinct roles or layers, such as human, agent, and artifact.

These are concept studies only. If one direction is selected, the canonical rationale in `docs/identity.md` should be updated to reference the chosen mark and its symbolism.
